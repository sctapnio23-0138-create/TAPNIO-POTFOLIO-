# TAPNIO Portfolio

A modern, responsive portfolio website built with HTML, CSS, and JavaScript.

## Features

✨ **Modern Design** - Clean and professional layout with gradient accents
📱 **Fully Responsive** - Looks great on desktop, tablet, and mobile devices
⚡ **Smooth Interactions** - Hover effects, animations, and smooth scrolling
📚 **Multiple Sections** - Hero, About, Skills, Projects, and Contact sections
💬 **Contact Form** - Built-in contact form with validation
🔗 **Social Links** - Links to your social media profiles
♿ **Accessible** - Semantic HTML and keyboard navigation support

## Sections Included

- **Header** - Navigation menu with smooth scrolling
- **Hero** - Eye-catching introduction with call-to-action button
- **About** - Personal introduction and background
- **Skills** - Showcase your technical skills in organized cards
- **Projects** - Featured projects with descriptions and tech tags
- **Contact** - Contact form and social media links
- **Footer** - Copyright information

## Customization

To personalize the portfolio, edit the following in `index.html`:

1. **Name & Title** - Replace "Your Name" with your actual name in the hero section and about section
2. **About Section** - Update your bio and background
3. **Skills** - Modify the skill cards to match your expertise
4. **Projects** - Update project titles, descriptions, and technologies
5. **Social Links** - Add your actual social media URLs
6. **Colors** - Change the gradient colors by modifying the hex codes (#667eea, #764ba2)

## Areas to Customize

- Line with "Your Name" → Replace with your name
- About section paragraph → Your bio
- Skill cards → Your technical skills
- Project cards → Your portfolio projects
- Social media links → Your profiles
- Contact area → Can add email or contact method

## How to Use

1. Open `index.html` in your web browser
2. Customize the content with your information
3. Add actual images to the portfolio image placeholder
4. Update social media URLs in the contact section
5. Deploy to GitHub Pages, Netlify, or your hosting provider
