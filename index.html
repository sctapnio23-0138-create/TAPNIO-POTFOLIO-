Midterm Lab 1: Task Data Cleaning and Modeling

In this activity, I performed data cleaning and preparation using Microsoft Excel on a Bike Sales dataset. The objective of the task was to transform raw and unorganized data into a clean, structured format that can be used for accurate analysis and decision-making.

First, I explored the dataset to identify possible issues such as duplicate entries, missing values, and inconsistent data. I expanded columns and reviewed each field carefully to detect errors that could affect the results of data analysis.

Next, I performed data cleaning by identifying duplicate records, particularly in the Sales Order ID column. I used conditional formatting to highlight duplicates and corrected incorrect entries. I also removed redundant rows to ensure that each record is unique and reliable.

After that, I checked for missing or blank values in the dataset. Using conditional formatting, I highlighted empty cells and filled them with appropriate values based on the context of the data. This step ensured completeness and avoided inaccurate analysis caused by missing information.

I also applied data transformation techniques such as "Text to Columns" to separate combined data fields. Specifically, I split the product description into multiple columns such as model, color, and size. This improved data organization and made it easier to analyze specific attributes.

Finally, after cleaning and transforming the dataset, I structured the data into a more organized format suitable for analysis and modeling. This includes preparing the data for a star schema model, where the main fact table contains measurable data such as sales and profit, and dimension tables contain descriptive information such as product, customer, date, and location.

Overall, this activity helped me understand the importance of data cleaning and preparation in ensuring data accuracy, consistency, and usability for better analysis and decision-making.
